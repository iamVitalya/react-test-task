## Для запуска проекта нужно:

### `yarn start`

Приложение запускаетется в режиме разработки \
Путь [http://localhost:3000](http://localhost:3000) для отображения фронта.

### `yarn json-server-start`

Приложение запускает fake REST API (Объязательно нужно для корректной работы фронта)\
Путь [http://localhost:3300](http://localhost:3300).

### `yarn build`

Для сборки проекта.
